DATA STRUCTURES
│
├── Primitive Data Structures
│   │
│   ├── Integer
│   │   • Stores whole numbers
│   │   • Drawback:
│   │      - Cannot store decimals
│   │
│   ├── Float / Double
│   │   • Stores decimal numbers
│   │   • Drawback:
│   │      - Precision errors may occur
│   │
│   ├── Character
│   │   • Stores one character
│   │   • Drawback:
│   │      - Stores only one character
│   │
│   └── Boolean
│       • Stores True or False
│       • Drawback:
│          - Only two possible values
│
└── Non-Primitive Data Structures
    │
    ├── Linear Data Structures
    │   │
    │   ├── Array
    │   │   • Fixed-size collection
    │   │   • Fast random access (O(1))
    │   │   • Drawbacks:
    │   │      - Fixed size
    │   │      - Slow insertion/deletion
    │   │
    │   ├── Vector
    │   │   • Dynamic array (resizable)
    │   │   • Rich STL functions
    │   │   • Fast random access (O(1))
    │   │   • Drawbacks:
    │   │      - Slight memory overhead
    │   │      - Resizing may require copying elements
    │   │
    │   ├── Linked List
    │   │   • Nodes connected by pointers
    │   │   • Dynamic size
    │   │   • Drawbacks:
    │   │      - No direct indexing
    │   │      - Extra memory for pointers
    │   │
    │   ├── Stack (LIFO)
    │   │   • Last In, First Out
    │   │   • Used in recursion, undo
    │   │   • Drawback:
    │   │      - Only top element is accessible
    │   │
    │   └── Queue (FIFO)
    │       • First In, First Out
    │       • Used in scheduling and buffering
    │       • Drawback:
    │          - Restricted access
    │
    └── Non-Linear Data Structures
        │
        ├── Tree
        │   • Parent-child hierarchy
        │   • Used in file systems and databases
        │   • Drawbacks:
        │      - Complex implementation
        │      - Can become inefficient if unbalanced
        │
        └── Graph
            • Collection of vertices and edges
            • Used in maps and social networks
            • Drawbacks:
               - Complex algorithms
               - Higher memory usage
